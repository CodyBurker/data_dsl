Syntax Guide:

            
    
                
  • Supported: `LOAD_CSV FILE "name"`, `KEEP_COLUMNS cols`, `PEEK`.
    • 
                
  • Other commands are parsed but not yet interpreted.
    • 
                
  • Piping: Use `THEN`. Comments: Start with `#`.
    • 
                
  • CSV Loading now uses a basic native parser. For complex CSVs (e.g., with commas in fields), issues might occur.
